# WebSites

Репозиторий для хранения всех созданных сайтов.

## Структура

Каждый сайт находится в отдельной папке в корне репозитория. Название папки соответствует названию сайта.

Пример:

```
WebSites/
├── NeyroMir/   # сайт про нейросети
│   └── ...     # файлы сайта
└── ...
```

## Ветки

- `main` — стабильная версия, только рабочие сайты.
- `dev` — ветка для разработки и тестирования новых сайтов/изменений перед переносом в `main`.
