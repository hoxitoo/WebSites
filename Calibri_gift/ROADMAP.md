# Roadmap — сайт «Колибри» (Calibri_gift)

Имиджевый новогодний лендинг для ТК «Колибри»: корпоративные подарки как забота
о сотрудниках. Купить с сайта нельзя — цель: захотеть каталог → личный контакт/бот.

**Стек:** Next.js 15 + TypeScript + Tailwind v4 + Motion (framer-motion)
**Ветка:** `dev` · **Заявки:** почта + Google Таблица · **ЦА:** HR, собственники,
директора, закупки, главбухи, профсоюзы (компании от ~300 сотрудников)

**Палитра:** тёмно-синий `#0E1526` / бордовый `#7A2438` / золото `#E8B968` /
тёплый белый `#F7F3EC` · **Шрифты:** Playfair Display + Manrope

## 📌 Решения и договорённости

- Сайт только под новогоднюю кампанию (мерч — второстепенно, позже)
- Цены и каталог НЕ публикуем — защита идей от конкурентов
- Тон: тёплый-эмоциональный, премиальный (не мультяшный) —
  «руководитель дарит не подарок, а признание вклада»
- Пуш в GitHub — по команде заказчика разработки
